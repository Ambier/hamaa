# 2016年11月02日

Hamaa已支持特性：

- models:
	- Sequetial()
- layers:
	- Dense
	- Activation
	- Convolution2D
	- MeanPooling2D
	- Flatten
- activations:
	- linear
	- sigmoid
	- tanh
	- relu
- initializations:
	- zeros
    - uniform
    - normal
    - glorot_normal
- objectives:
	- mse/mean_square_error
    - categorical_crossentropy
- optimizers:
	- SGD
